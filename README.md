# bid-strategy-v2

낙찰데이터 기반 Streamlit 입찰전략 분석 시스템입니다.

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 1차 기능

- 엑셀 데이터 업로드
- 날짜/금액/사정률 기본 정제
- 전체 현황 대시보드
- 기관별 분석
- 업체별 분석
- 업체수별 경쟁강도 분석
- 기초 추천 투찰률 산출
- 리스크 요약

## 권장 데이터 컬럼

- 공고명
- 발주기관
- 기초금액
- 예정가격
- 낙찰업체
- 사정률
- 개찰일
- 업체수
- 업종
